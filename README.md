# Goodbye Dry Eye

## Usage
1. Set your scheduling tool to run this script at regular intervals.
2. When it starts, take a moment to do a deep blink or a simple eye exercise.
3. Your eyes will thank you - enjoy feeling more relaxed and refreshed!

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/danildavletov/goodbye-dry-eye.git
   ```
2. Navigate to the project directory:
   ```bash
   cd goodbye-dry-eye
   ```
3. Install required dependencies (pygame):
   ```bash
   pip install -r requirements.txt
   ```
4. Run the script:
   ```bash
   python blink.py
   ```
